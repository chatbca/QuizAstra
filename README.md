QuizAstra – Android Quiz Application

Developed an enterprise-grade Android quiz ecosystem consisting of a User App for learners and an Admin Panel for content and user management. Built with Java 8, Firebase Realtime Database, and Material Design 3, the system supports real-time multiplayer battles, global leaderboards, and secure data handling.
1.User Application
Secure Login & Registration with email, password strength checks, and profile pictures.
Personalized Dashboard with quiz categories, stats, and quick actions.
Dynamic Quiz Engine with timers, progress indicators, and anti-cheating (screenshot prevention, copy-block).
Real-time Multiplayer Battles with synchronized countdowns and winner determination.
Leaderboards (global + category-wise) with avatars and fair tie-breaking logic.
Detailed Results Analysis (score breakdown, question review, badges, history).
User Profile Management (edit info, update password, track achievements).
2.Admin Panel
Secure Admin Login with credential validation and session management.
Admin Dashboard showing real-time analytics (users, questions, categories, attempts).
Question Management – Create, update, delete, and filter questions by category.
Category Management with relationship checks and cascading delete.
Leaderboard Viewer for performance monitoring and rankings.
User Details View – Track individual user performance and manage profiles.
Bulk User Management – Search, filter, and delete multiple users at once.
3. Technical Features
Firebase Realtime Database for live synchronization (quizzes, battles, leaderboards).
Offline-first support – quizzes and stats cached locally, syncs when online.
Data Security – encrypted transmission, validation checks, anti-cheating.
Scalability – architecture supports thousands of users simultaneously.
Material Design 3 UI/UX – modern, intuitive, and responsive interface.
